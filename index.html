CDC 7 Words

Recently, the Washington Post revealed that the Trump Administration chose to censor the CDC from using certain words to communicate their budgetary needs to Congress. This has caused an uproar across twitter as #CDC7words has become a trending topic.

But what does this censorship actually mean for the CDC? How has the CDC used these words in the past? Will the CDC be able to communicate clearly regarding the complex clinical topics that it discusses in its budget proposals?

Figuring that out is pretty hard without doing some data mining. Each CDC Budget Justification document is hundreds of pages long, and there are more than 10 years worth of these documents to do through. So I wrote some code to automated the process of searching through the 10k+ pages, which is now available on github. This website is automatically generated as result of that data mining.

Using this tool, you can see for yourself how difficult it is going to be for the CDC to communicate clearly with Congress without the use of these terms, which have specific clinical meanings that matter for CDC strategy.

It is not all bad news. While some of these terms are used constantly to reveal facts about clinical topics that would be hard to describe without them. The word "vulnerable" is not been previously used by the CDC in these documents. (although I did not search for synonyms or even different words with the same root, like "vulnerability", and I also did not do a comprehensive search for uses of this term at the CDC, but outside these documents) . This does not mean "vulnerable" is not an important clinical term, it just means that the CDC should be able to discuss funding intelligently without using the term.

The 7 censored words

  • diversity
  • entitlement
  • evidence
  • science-based
  • trangender
  • fetal
  • vulnerable (not ever found)
